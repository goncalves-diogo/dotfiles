[#](#) Dotfiles

Description of my current dotfiles, and how to install it

## General configuration

Current OS: Ubuntu 22.04

## Installation

1. Run `install.sh`
   Or
1. Install nix & home-manager
1. Install stow
1. `$ stow home-manager`
1. `home-manager switch`
